Project database web application for the ECE 461L team project.

##
Frontend Quick Start:
1. ```git clone https://github.com/RowanB141/ProjectDatabaseWebApp.git```
2. Open cmd
3. Navigate to frontend folder
4. Run ```npm install```
5. Run ```npm run dev```
